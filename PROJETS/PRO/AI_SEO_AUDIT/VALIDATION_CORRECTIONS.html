Problèmes identifiés

❌ Problème 1 : Résultats en anglais

Cause : Le système prompt envoyé à OpenAI est hardcodé en anglais dans ai_provider.py (lignes 94-96)

Impact : ChatGPT répond en anglais même quand l'utilisateur a demandé du français

Fichiers concernés :

  • src/core/interface/ai_provider.py
  • src/orchestrator/agents/audit_agent.py

❌ Problème 2 : Descriptions incompréhensibles

Cause : Les textes générés utilisent des placeholders non remplis comme [expertise principale], [X années]

Impact : Les recommendations affichent du texte générique avec des crochets, peu exploitable

Fichiers concernés :

  • src/orchestrator/agents/analyze_agent.py
  • src/orchestrator/agents/generate_agent.py

Solutions proposées

✅ Solution 1 : Gestion multilingue des prompts (30 min)

  • Créer module src/core/config/prompts.py avec prompts FR/EN/ES/DE
  • Modifier ai_provider.py pour accepter paramètre language
  • Passer la langue depuis audit_service.py → orchestrator → agents → provider
  • L'IA répondra dans la langue demandée

✅ Solution 2 : Descriptions claires et exploitables (20 min)

  • Réécrire les descriptions des gaps en français clair
  • Ajouter contexte : "Pourquoi c'est important", "Ce que ça implique concrètement"
  • Exemple : "Votre entreprise n'apparaît que dans 2 requêtes sur 3. Vos concurrents sont plus visibles."

✅ Solution 3 : Génération intelligente de contenu (30 min)

  • Remplacer les placeholders [expertise principale] par du contenu réel
  • Utiliser les données de l'analyse (secteur, concurrents) pour personnaliser
  • Générer des exemples concrets au lieu de textes génériques
