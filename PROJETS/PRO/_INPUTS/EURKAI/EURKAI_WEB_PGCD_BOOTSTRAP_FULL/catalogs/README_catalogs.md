# Catalogues Eurkai (PGCD)

Tous les catalogues vivent ici :

- `objects/`  : types d’objets, schémas, signatures
- `prompts/`  : prompts, métaprompts, rôles, missions
- `modules/`  : modules réutilisables (pages, sections, blocks…)
- `domains/`  : spécialisations par domaine (éducation, SaaS, coaching…)

Ces fichiers sont *vivants* : l’IA et toi les enrichirez progressivement.
