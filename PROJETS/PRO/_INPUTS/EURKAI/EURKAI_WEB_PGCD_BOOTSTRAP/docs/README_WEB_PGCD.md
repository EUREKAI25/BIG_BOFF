# Eurkai — WEB_PGCD Bootstrap

Ce répertoire contient le socle local minimal pour :

- tester le cockpit fractal
- centraliser tous les catalogues
- stabiliser ERK + MetaSchema
- préparer les tests T1/T2/T3

Ordre suggéré :
1. Lire `erk/ERK_V1_NOTICE.md`
2. Vérifier le MetaSchema dans `metaschema/`
3. Copier `object_fractal_tool_v4.html` dans `cockpit/`
4. Ouvrir le cockpit et pointer vers `tests/data/objects_T1.json`
5. Ensuite seulement : étendre les catalogues.
