# TEST 02 — Tags, Catégories & Alias

Checklist :

- [ ] La sidebar des tags affiche tous les tags par ordre alphabétique.
- [ ] Le scroll fonctionne avec une longue liste de tags.
- [ ] Je peux drag & drop un tag sur un ObjectType :
  - [ ] le tag apparaît dans les métadonnées de cet ObjectType.
- [ ] Clic droit sur un tag :
  - [ ] je peux activer/désactiver `isCategory`,
  - [ ] si `isCategory` est true, je peux définir une `parentCategory`.
- [ ] Sur la vue de détail d’un ObjectType ou vecteur :
  - [ ] je vois la liste des alias existants,
  - [ ] je peux ajouter un alias,
  - [ ] l’alias est enregistré dans le JSON en mémoire.
