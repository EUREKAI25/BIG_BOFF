
# K1 — Site de l’agence & Portail Eurkai

## Objectif
Définir le site / portail public d’Eurkai :
- présentation de la vision,
- description de l’offre (projets, services, modules),
- onboarding des utilisateurs (compte, accès, démo),
- connexion avec les instances générées par Eurkai.

## Ce que tu dois produire
- La structure du portail (ObjectTypes côté fractale + modules côté front).
- Les grandes sections (Home, Pourquoi, Comment, Offres, Projets, Labo, Contact…).
- La stratégie d’intégration avec les projets créés par Eurkai (vitrines, cas d’usage, démonstrations).

## Contraintes
- Rester aligné avec le langage visuel minimaliste et la logique fractale (navigation par tags / vecteurs).
