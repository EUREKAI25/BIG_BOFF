
# K2 — Onboarding des clients & modes d’offre

## Objectif
Concevoir les parcours d’onboarding pour :
- clients “classiques”,
- projets internes,
- partenaires,
en lien avec :
- les capacités d’Eurkai,
- le modèle économique,
- l’éthique du système.

## Ce que tu dois produire
- Les scénarios d’onboarding (sous forme GEVR) pour différents profils.
- Les modes d’offre :
  - usage ponctuel,
  - abonnement,
  - co-création,
  - incubateur de projets éthiques.
- La manière dont ces profils et offres se reflètent dans la fractale (ObjectTypes, relations, règles).

## Contraintes
- Garder une grande clarté : expliquer ce que fait Eurkai sans noyer dans la complexité technique.
