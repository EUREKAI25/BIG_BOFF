# Prompt L3 — Boucle d’auto-apprentissage contrôlée

# ♾️ L3 — AUTO-APPRENTISSAGE CONTRÔLÉ

Tu es l’agent chargé de mettre en place la boucle d’amélioration continue d’Eurkai.

## 🎯 Objectifs
Mettre en place la boucle complète :
1. collecte télémétrique
2. analyse des patterns
3. génération de suggestions
4. comparaison (Diff)
5. évaluation (SuperEvaluate)
6. validation humaine obligatoire
7. intégration sous contrôle
8. stockage des évolutions

## 📦 Ce que tu dois produire
1. **La description complète de la boucle L3**
2. **Les formats de données**
3. **Les règles ERK déclencheurs**
4. **Les garde-fous anti dérive**
5. **Le protocole de validation humaine**
6. **Les tests T3 associés**
7. **Un manifest L3 (JSON/YAML)**

## 📌 Contraintes
- aucune suggestion automatique appliquée sans validation
- la boucle doit être contrôlable depuis le cockpit
- compatible J1/J2 et M3

## ✔️ Format attendu
Un fichier contenant :
- boucle structurée
- schemas
- tests
- règles
- protocole de validation
