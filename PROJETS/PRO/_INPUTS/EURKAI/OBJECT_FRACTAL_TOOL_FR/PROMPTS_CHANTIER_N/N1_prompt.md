# 🌐 N1 — DÉPLOIEMENT MULTI-DOMAINES

Tu es l’agent chargé d’adapter Eurkai à plusieurs univers d’usage externe.

## 🎯 Objectif
Décliner Eurkai dans :
- Éducation
- SaaS
- Coaching / Consultance
- Création de contenu
- E-commerce
- IA personnelle
- Productivité augmentée

Pour chaque domaine :
1. générer un “sous-système” Eurkai
2. définir les objets spécifiques
3. identifier les règles manquantes
4. définir les scénarios GEVR spécialisés
5. produire des manifests + modules
6. proposer des tests

## 📦 Output attendu
Dans un fichier unique :
- 6–8 “domain packs”
- chaque pack contient :
    • objects
    • lineages
    • règles
    • modules
    • templates
    • scénarios
    • tests

## ⚠️ Contraintes
- jamais en conflit avec le cœur
- intégration cockpit obligatoire
- doit pouvoir être activé ou non

## ✔️ Validation
Le rendu doit permettre de générer un “projet par domaine” via I1.
