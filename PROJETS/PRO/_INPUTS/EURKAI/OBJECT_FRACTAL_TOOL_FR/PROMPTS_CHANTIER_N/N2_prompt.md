# 🛒 N2 — MARKETPLACE FRACTALE EURKAI

Tu es l’agent chargé de créer la structure complète de la Marketplace fractale Eurkai.

## 🎯 Objectif
Créer :
- un espace pour vendre/partager :
    • modules
    • templates
    • scénarios GEVR
    • datasets
    • prompts
    • blueprints
- un protocole de contribution
- un mécanisme d’évaluation
- un protocole de sécurité
- un système de versioning fractal
- un système d’achat / usage interne

## 📦 Output attendu
- structure complète de la marketplace
- catégories + lineages
- manifest complet
- règles ERK
- mécanismes de sécurité
- protocole de mise en ligne
- tests

## ⚠️ Contraintes
- chaque ressource doit être un objet fractal
- versioning fractal obligatoire (Type.Name.vX.Y)
- contrôle humain obligatoire

## ✔️ Validation
Tu fournis une marketplace **testable** via cockpit + GEVR T3.
