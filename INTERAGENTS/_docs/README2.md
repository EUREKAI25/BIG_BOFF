# EURKAI_COCKPIT

Ce ZIP contient des **CHANTIERS**. Chaque chantier = 1 répertoire avec:
- `OUTPUTS/` (réservé aux livrables générés)
- un prompt `EURKAI_COCKPIT_prompt_CXX.md` à envoyer à l’agent responsable.

Règles générales:
- livrables **testés/validés**
- inclure un **CLI** pour intégration simplifiée (import/export/run)
- local-first, pas de cloud obligatoire
