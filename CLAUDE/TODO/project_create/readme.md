# Project_Create

## Vision
Système multi-agents à rôles étanches.

## Agents
- Split : découpe brief en 4-6 étapes
- Triage : classe MVP ou NEXT  
- Build : génère code + test.sh
- QA : valide (PASS/FAIL)
- Backlog : prépare version suivante

## Règles
1. Agents isolés
2. Max 6 étapes par version
3. MVP first toujours
4. Tests obligatoires
5. Tag stable avant nouvelle version
